## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Bitsnbytes14/ahmad-netflix-portfolio.git
cd ahmad-netflix-portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

## 🌐 Deployment

This project is deployed on **Vercel**.

To deploy your own:

3. Click Deploy 🚀
